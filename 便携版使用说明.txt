========================================
   SPR Analysis 便携版 v1.0
========================================

由于exe打包遇到兼容性问题，此为便携版方案。
此版本包含完整Python环境，100%可用。

----------------------------------------
  使用方法
----------------------------------------

1. 解压整个文件夹到任意位置

2. 双击 "启动程序.bat" 启动软件

3. 等待几秒，程序窗口会自动打开

就这么简单！

----------------------------------------
  文件说明
----------------------------------------

启动程序.bat      ← 双击这个文件启动
.venv\            ← Python环境（请勿删除）
app_full.py       ← 主程序
src\              ← 程序源代码
XlementFitting\   ← 拟合算法库

----------------------------------------
  系统要求
----------------------------------------

- Windows 10/11 (64位)
- 2GB+ 内存
- 1GB 磁盘空间

无需安装Python或其他依赖！

----------------------------------------
  主要功能
----------------------------------------

✓ 数据导入/导出 (Excel/CSV/JSON)
✓ 数据拟合 (单个/批量)
✓ 图表显示和编辑
✓ 数据对比分析
✓ 数据血缘可视化
✓ 操作撤销/重做
✓ 会话保存/加载

----------------------------------------
  快捷键
----------------------------------------

Ctrl+O  导入数据
Ctrl+S  保存会话
F5      数据拟合
Ctrl+M  数据对比
Ctrl+L  血缘图
Ctrl+Z  撤销
Ctrl+Y  重做

----------------------------------------
  常见问题
----------------------------------------

Q: 为什么不是单个exe文件？
A: 为了确保100%兼容性和稳定性，
   采用便携版方案。

Q: 可以复制到U盘吗？
A: 可以！整个文件夹都可以复制。

Q: 需要安装吗？
A: 不需要！解压后直接使用。

Q: 启动失败怎么办？
A: 检查是否解压完整，确保.venv文件夹存在。

  分发说明

- 包含 .venv\ 文件夹

- 包含 启动程序.bat

用户收到后：
1. 解压
2. 双击 启动程序.bat
3. 开始使用
