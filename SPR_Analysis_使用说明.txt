========================================
    SPR Analysis - 软件使用说明
========================================

版本: 1.0
发布日期: 2025年1月23日

----------------------------------------
  安装说明
----------------------------------------

本软件为绿色版，无需安装：

1. 解压 SPR_Analysis 文件夹到任意位置

2. 打开 SPR_Analysis 文件夹

3. 双击 SPR_Analysis.exe 启动程序

注意：
- 首次启动可能需要5-10秒
- 请勿删除 _internal 文件夹中的文件
- 建议将整个文件夹放在非系统盘

----------------------------------------
  快速入门
----------------------------------------

第一步：导入数据
  方法1: 文件 → 导入数据 (Ctrl+O)
  方法2: 点击工具栏 📁 图标
  
  支持格式: Excel (.xlsx), CSV (.csv), JSON (.json)

第二步：查看数据
  - 在左侧项目树中点击数据节点
  - 中间区域显示数据表格
  - 右侧显示数据详情

第三步：数据拟合
  - 选中数据节点
  - 按 F5 或右键 → 拟合分析
  - 选择拟合方法 → 开始拟合
  - 查看生成的图表和结果

第四步：导出结果
  - 右键图表节点 → 导出图表
  - 右键数据节点 → 导出数据

----------------------------------------
  常见问题 FAQ
----------------------------------------

Q1: 为什么首次启动较慢？
A: 首次启动需要初始化，属于正常现象。后续启动会快很多。

Q2: 如何批量拟合多个数据？
A: 在项目树中，按住 Ctrl 键多选数据 → 右键 → 批量拟合

Q3: 如何对比多个数据？
A: 方法1: Ctrl+M 打开对比窗口，然后在项目树中选择数据 → 右键 → 添加到对比
   方法2: 多选数据 → 右键 → 添加到对比

Q4: 拟合失败怎么办？
A: 检查数据格式是否正确（需要两列数值数据：X和Y）

Q5: 如何保存当前工作？
A: 文件 → 保存会话 (Ctrl+S)，会保存所有数据、图表和结果

Q6: 图表如何调整样式？
A: 右键图表节点 → 编辑样式，或在图表区域右键 → 编辑样式

Q7: 如何查看操作历史？
A: 编辑 → 操作历史，可以查看所有操作记录

Q8: 数据血缘图有什么用？
A: 显示数据之间的关系，帮助理解数据的来源和处理流程

----------------------------------------
  数据格式要求
----------------------------------------

Excel 文件:
  - 第一行可以是表头（可选）
  - 至少两列数值数据
  - 第一列为 X 值，第二列为 Y 值

CSV 文件:
  - 逗号分隔
  - 格式同 Excel

JSON 文件:
  - 需符合特定格式（请参考示例文件）

----------------------------------------
  系统要求
----------------------------------------

操作系统: Windows 10/11 (64位)
内存: 2GB 以上
硬盘: 200MB 可用空间
其他: 无需安装 Python 或其他依赖

----------------------------------------
  注意事项
----------------------------------------

1. 请定期保存会话，避免数据丢失

2. 大数据量拟合可能需要较长时间，请耐心等待

3. 批量拟合时会使用多线程，可能占用较多CPU

4. 导出的图表默认保存在软件同目录下的 exports 文件夹

5. 会话文件（.sprx）包含所有工作状态，建议定期备份

6. 如遇到问题，可查看状态栏的提示信息
