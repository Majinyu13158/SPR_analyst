========================================
    SPR Analysis v1.0 - 使用说明
========================================

快速开始：

1. 双击 "启动SPR.bat" 启动程序

2. 导入数据:
   文件 → 导入数据 (Ctrl+O)
   支持 Excel, CSV, JSON 格式

3. 数据拟合:
   选择数据 → 按 F5 或右键 → 拟合分析

4. 查看结果:
   点击图表/结果节点

========================================

主要功能：

✓ 数据导入/导出 (Excel/CSV/JSON)
✓ 数据拟合 (单个/批量/多线程)
✓ 图表编辑和导出 (PNG/PDF/SVG/JPG)
✓ 数据对比分析 (Ctrl+M)
✓ 数据血缘可视化 (Ctrl+L)
✓ 操作撤销/重做 (Ctrl+Z/Y)
✓ 项目树搜索 (Ctrl+F)
✓ 样式编辑器
✓ 会话保存/加载 (Ctrl+S)

========================================

快捷键：

Ctrl+O  导入数据
Ctrl+S  保存会话
F5      数据拟合
Ctrl+M  数据对比
Ctrl+L  血缘图
Ctrl+Z  撤销
Ctrl+Y  重做
Ctrl+F  搜索

========================================

系统要求：

- Windows 10/11 (64位)
- 2GB+ 内存
- 无需安装Python

========================================

注意事项：

- 首次启动需要几秒钟
- 请勿删除 .venv 文件夹
- 可以复制到U盘使用
